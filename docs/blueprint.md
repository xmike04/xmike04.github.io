# **App Name**: Marin Insights

## Core Features:

- Animated Portfolio Timeline: Interactive timeline showcasing key projects and accomplishments, with animated transitions and scroll-triggered effects.
- Interactive Mini-Game: Embedded mini-game that reflects AI/ML concepts (e.g., a simple neural network visualizer) to engage visitors.
- Dynamic Resume: Dynamic resume display, rendered directly from JSON data. Toggle button to show and hide less relevant info.
- AI-Powered Chatbot: AI chatbot assistant that provides positive, tailored responses to employer inquiries using Gemini tool, based on the provided resume.
- Prominent Contact Information: Clear calls to action that highlights contact information for potential employers.
- Responsive Design: Fully responsive layout adapting to various screen sizes and devices for optimal viewing experience.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2), inspired by data visualization and trustworthiness.
- Background color: Very light gray (#F0F4F7), providing a clean and modern backdrop for content.
- Accent color: Bright purple (#9C27B0) for highlighting interactive elements and CTAs.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Minimalist, custom-designed icons representing skills and projects.
- Clean and modern layout with a clear hierarchy, ensuring ease of navigation and content consumption.
- Subtle animations and transitions on scroll and hover, enhancing interactivity and visual appeal.
